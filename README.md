# How to run in JS

```bash
    cd js
```

```bash
    npm run start
```

# How to run tests in JS

```bash
    cd js
```

```bash
    npm run test
```

# How to run in PY

```bash
    cd py
```

```bash
    python main.py
```

# How to run tests in PY

```bash
    cd py
```

```bash
    pytest -v 
```
